<h2>Operating Instructions</h2>
After you build, remember to source the proper devel folder…
```cpp
source ~/catkin_ws/devel/setup.bash
```
And then run the launch file…
```cpp
roslaunch sm_dance_bot sm_dance_bot.launch
```
